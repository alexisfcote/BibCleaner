BibCleaner is a utility to help clean up a .bib text file by removing unwanted entries.

Usage:
-Load your .bib file
-Select entrie you whish to delete
-Apply to delete them form the file

Note:
BibClean will not delete entries under the misc type so.
You can manage the default entries to delete by adding the entry keys to de default file. This file will be created on first run of the program:w
